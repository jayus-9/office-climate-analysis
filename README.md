# Анализ данных о климате в офисах
Анализ данных о климате в офисах: EDA, статистические тесты, регрессионное моделирование и визуализация. Проект демонстрирует навыки работы с Python и основными библиотеками для анализа данных.
## Описание проекта
Проект является полным анализом датасета о климатических условиях в офисах.

Цель: проанализировать полученные данные и предоставить выводы о том, в какой тип климатического оборудования стоит инвестировать больше, чтобы повысить удовлетворенность клиентов созданными условиями в зависимости от региона и на что стоит обратить внимание при развитии бизнеса в разных регионах.

Данные: CSV-файл с наблюдениями (Climat.csv).

## Цели
- Предобработка данных (обработка пропусков, аномалий).
- Exploratory Data Analysis (EDA) с визуализациями.
- Статистические тесты (корреляция, гипотезы).
- Регрессионное моделирование (линейная регрессия для предсказания температуры).
- Выводы.

## Используемые инструменты
- Python: pandas, numpy, matplotlib, seaborn, scikit-learn, scipy.
